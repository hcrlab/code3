# Code3
Code3 is a robot programming system for mobile manipulators.

Documentation on how to build, deploy, and use Code3 is forthcoming.

In the meantime, relevant source code links are listed here:
- CustomActions: [PR2/pr2_pbd](https://github.com/PR2/pr2_pbd)
- CustomLandmarks: [jstnhuang/rapid](https://github.com/jstnhuang/rapid)
- CodeIt: [hcrlab/code_it](https://github.com/hcrlab/code_it)

Supported robots include:
- [PR2](https://github.com/hcrlab/code_it_pr2)
- [Turtlebot](https://github.com/hcrlab/code_it_turtlebot) (CodeIt only)
- Support for the Fetch is forthcoming
